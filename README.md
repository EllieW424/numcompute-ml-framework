# NumCompute ML Framework

A lightweight machine learning framework built from scratch using Python and NumPy.

## Project Overview

This project implements core machine learning infrastructure without relying on external ML libraries such as Scikit-Learn or PyTorch.

The framework supports:

- CSV data loading
- Streaming data processing
- Data preprocessing
- Feature transformation
- Pipeline abstraction
- Numerical computation using NumPy
