# NumCompute ML Framework

A lightweight machine learning framework built from scratch using Python and NumPy.

---

## Project Overview

NumCompute is a custom machine learning framework developed as part of the Programming for Artificial Intelligence coursework.

The project aims to reproduce core functionality commonly found in machine learning libraries while focusing on algorithmic understanding, numerical computation, and software engineering principles.

The framework was implemented using only Python and NumPy without relying on external machine learning libraries such as Scikit-Learn or PyTorch.

---

## Features

* CSV data loading and parsing
* Streaming data processing
* Data preprocessing utilities
* Feature transformation pipeline
* Modular pipeline abstraction
* Numerical computation with NumPy
